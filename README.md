# Here are your Instructions

How to start the backend:
1. On the terminal enter- cd backend, and press Enter
2. Then enter- node server.js, and press Enter
3. The terminal will show- Server is running on port 8001

How to start the frontend:
1. On a different terminal enter- cd frontend, and press Enter
2. Then enter- yarn start, and press Enter
3. This will open your default browser and display the website.
